# 🎵 Global Music Learner
An interactive web application designed to help language learners practice listening skills through music.

## 🚀 Features
* **Customizable:** Paste any YouTube music video link.
* **Interactive:** Unscramble lyrics in real-time.
* **Gamified:** Start with 1000 points and try to keep them by avoiding mistakes!

## 🛠️ How to Use
1. Enter a YouTube URL of a song you want to learn.
2. Type a line of lyrics from that song.
3. Click **Create Activity**.
4. Click the words in the correct order as you listen to the music!
